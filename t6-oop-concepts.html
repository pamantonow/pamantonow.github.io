Instance variables are use in a class this very similar as the class variables except that their values are local to specific instance of an object. For example if a class contains an instance variable called @answer, if one instance of the object changes the current value of @answer the change is local to only the object that made the change. Other objects of the same class have their own local copies of the variable which are independent of changes made in any other objects. 
      class Questions

       def initialize(answer)
       @answer= answer
       end
     end

I am creating a class called questions, and defining a method called initialize and passing and argument then below that I am creating the instace of a variable by putting the @symbol.

Class variables are very much like instance variables but this is shared among this variables, this means that only one variable value exists for all objects instantiated from this class. This being said that if one object instance changes the value of the variable, that new value will essentially change for all other object instances. Class variables are declared by using @@ symbol.
