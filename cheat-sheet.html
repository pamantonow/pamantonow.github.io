Sort_by

Really similar as sort but sort_by will only take one argument and it can only be used with a block while sort can be used with or without a block. 
        aray = ["ana","pedro","juan"]
        array.sort_by { |word| word.length}

        #=> ["ana","juan","pedro"]

Collect

Returns a new array with the results of running block once for every element in enum. In other words it selects and modify an item. Collect will run your block and give you an object filled with whatever your block returned each time. 
        array = [2,3,4,5,6]
        array.collect {|num| num * 2}

        #=> [4,6,8,10,12]
